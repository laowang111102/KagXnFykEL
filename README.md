# 旅游信息管理系统 java807

本项目是一个基于 Spring Boot 和 Vue 的前后端分离的旅游信息管理系统，附有详尽的9000字文档。

## 技术栈

- **前端语言：** Vue
- **后端语言：** Java, Spring Boot
- **主要技术：**
  - Java
  - Spring Boot
  - MyBatis
  - MySQL
  - Vue

## 功能模块

### 前台

- 展示旅游新闻
- 景区信息展示
- 美食信息展示
- 旅游线路展示
- 在线留言功能

### 后台管理员

- 账号管理
- 地区管理
- 景点信息管理
- 地方美食管理
- 旅游线路管理
- 订单信息管理
- 新闻管理
- 系统管理

### 普通用户

- 旅游线路查看
- 订单信息管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 前端：Vue.js 兼容的浏览器
- 后端：JDK 1.8 或更高版本，Spring Boot 运行环境

## 部署步骤

（具体部署步骤请参考9000字项目文档）

1. 配置数据库环境（MySQL）
2. 部署后端服务（Spring Boot）
3. 部署前端服务（Vue.js）
4. 配置服务器
5. 启动服务
